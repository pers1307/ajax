.ajaxError()
Документация      Примеры

- Выполняется после выполнения неудачного Ajax запроса.

Принимаемые аргументы:

request
Принимает объект XMLHttpRequest.
settings
Принимает настройки.
(.ajaxError(function(request, settings){ ... })

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ $.ajax({ type: "POST", url: "test1.php", data: ({les : 3}), }); }); $("#buttest").ajaxError()(function(){ alert("Неудача"); });
