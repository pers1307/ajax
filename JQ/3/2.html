.ajaxComplete()
Документация      Примеры

- Выполняется каждый раз после выполнения Ajax запроса.

Принимаемые аргументы:

request
Принимает объект XMLHttpRequest.
settings
Принимает настройки.
(.ajaxComplete(function(request, settings){ ... })

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ $.ajax({ type: "POST", url: "test.php", data: ({les : 2}), }); }); $("#buttest").ajaxComplete(function(){ alert("Запрос завершен"); });
