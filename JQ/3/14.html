.post()
Документация      Примеры

- Загружает данные с сервера используя HTTP POST request.

Принимаемые аргументы: .load(url, [data], [func])

url
URL запроса (куда пойдет запрос).
[data]
Данные которые будут отправлены на сервер.
[func]
Функция которая будет вызвана после успешного получения данных.
[type]
Тип данных которые вернутся (: "xml", "html", "script", "json", "jsonp", "text").

PHP:

$a = $_POST["a"]; $b = $_POST["b"]; echo "Присланные данные a=".$a.", b= ".$b;

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ var a = $("#text1").val(); var b = $("#text2").val(); $.post( "test.php", {les : 14, a : a, b : b}, function(data){alert(data)} ) });
