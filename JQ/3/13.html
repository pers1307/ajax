.load()
Документация      Примеры

- Загружает данные с сервера и помещает возвращаемые данные HTML в материнский елемент.

Принимаемые аргументы: .load(url, [data], [func])

url
URL запроса (куда пойдет запрос).
[data]
Данные которые будут отправлены на сервер.
[func]
Функция которая будет вызвана после успешного получения данных.

PHP:

if (is_numeric($_POST["a"])) { echo "Это <strong>число</strong>"; } else { echo "Это <strong>символ</strong>"; }

HTML:

<button id="buttest">Отправить Ajax запрос</button> <div id="exmpleload"></div>

jQuery:

$("#buttest").click(function(){ var text = $("#text1").val(); var w = $("#exmpleload"); w.load('test.php', {a : text}, alert("Загружено")) });
