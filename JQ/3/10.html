.get()
Документация      Примеры

- Загружает данные с сервера используя HTTP GET request.

Принимаемые аргументы: .get(url, [data], [func], [type])

url
URL запроса (куда пойдет запрос).
[data]
Данные которые будут отправлены на сервер.
[func]
Функция которая будет вызвана после успешного получения данных.
[type]
Тип данных которые вернутся (: "xml", "html", "script", "json", "jsonp", "text").

PHP:

$a = $_GET["a"]; $b = $_GET["b"]; echo "Присланные данные a=".$a.", b= ".$b;

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ var a = $("#text1").val(); var b = $("#text2").val(); $.get( "test.php", {a: a, b: b}, function(data){alert(data)} ) });
