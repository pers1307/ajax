.ajaxSetup()
Документация      Примеры

- Установка настроек для $.ajax().

Принимаемые аргументы:

async
Тип boolean (true, false), по умолчанию true. При значении логическом значении true все запросы отсылаются асинхронно. Если значение false то запрос осуществляется синхронно, во время синхронного запроса браузер может быть заблокирован.
beforeSend
Функця, которая выпольняется перед передачей запроса.
complete
Функция, которая выполнится после завершения запроса.
data
Данные, которые передаются на сервер. Если данные не являются строкой, то они конвертируются в строку запроса. Для запросов типа GET данные прикрепляются к URL. Объект должен состоять из пар ключ/значение. Если в значении массив, то jQuery упорядочивает значения, принадлежащие одному ключу, например, {foo:["bar1", "bar2"]} превращается в &foo=bar1&foo=bar2.
error
Функция выполняется в случае неудачного завершения запроса.
success
Функция выполненяется в случае успешного завершения запроса
type
тип запроса - POST либо GET. По умолчанию - GET.
url
url запроса. По умолчанию - текущая страница.

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").ajaxSetup({ type: "POST", url: "test1.php", data: ({les : 5}) }); $("#buttest").ajax({ data: myData });
