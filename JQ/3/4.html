.ajaxSend()
Документация      Примеры

- Выполняется после отправления Ajax запроса.

Принимаемые аргументы:

request
Принимает объект XMLHttpRequest.
settings
Принимает настройки.
(.ajaxSend(function(request, settings){ ... })

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ $.ajax({ type: "POST", url: "test.php", data: ({les : 4}) }); }); $("#buttest").ajaxSend(function(){ alert("Запрос отправлен"); });
