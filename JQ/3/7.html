.ajaxStop()
Документация      Примеры

- Функция выполняется каждый раз при завершении Ajax запроса.

Принимаемые аргументы:

request
Принимает объект XMLHttpRequest.
settings
Принимает настройки.
(.ajaxStop(function(request, settings){ ... })

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ $.ajax({ type: "POST", url: "test.php", data: ({les : 7}), }); }); $("#buttest").ajaxStop(function(){ alert("Запрос Завершен"); });
