.getScript()
Документация      Примеры

- Загружает JavaScript файл с сервера используя HTTP GET request, и выполняет его.

Принимаемые аргументы: .get(url, [func])

url
URL запроса (куда пойдет запрос).
[func]
Функция которая будет вызвана после успешного получения данных.

JavaScript:

document.bgColor = 'blue'; alert("this is JavaScript");

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ $.getScript( "test.js" ) });
