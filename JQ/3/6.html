.ajaxStart()
Документация      Примеры

- Функция выполняется каждый раз начале Ajax запроса.

Принимаемые аргументы:

request
Принимает объект XMLHttpRequest.
settings
Принимает настройки.
(.ajaxStart(function(request, settings){ ... })

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ $.ajax({ type: "POST", url: "test.php", data: ({les : 6}), }); }); $("#buttest").ajaxStart(function(){ alert("Запрос Стартовал"); });
