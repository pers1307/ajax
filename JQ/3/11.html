.getJSON()
Документация      Примеры

- Загружает данные в формате JSON с сервера используя HTTP GET request.

Принимаемые аргументы: .getJSON(url, [data], [func], [type])

url
URL запроса (куда пойдет запрос).
[data]
Данные которые будут отправлены на сервер.
[func]
Функция которая будет вызвана после успешного получения данных.
[type]
Тип данных которые вернутся (: "xml", "html", "script", "json", "jsonp", "text").

JSON:

{"id":"1","name":"ivan","country":"Russia","office":["yandex","management"]}

HTML:

<button id="buttest">Отправить Ajax запрос</button>

jQuery:

$("#buttest").click(function(){ $.getJSON( "test.php", function(data){alert(data)} ) });
